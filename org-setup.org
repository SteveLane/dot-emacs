* Introduction

This file provides details on my current =org= setup. It is intended as a literate programming approach to structure my =org= setup. It may work, or it may not (!) but I thought I'd give it a go.

If you want to use some (all) of this setup, you'll need to change directories appropriately. You can tangle this file to Emacs lisp using =C-c C-v C-t=, which will create org-setup.el in the same folder as this file. You can then source this in your init.el file.

This setup has been greatly influenced by [[http://cachestocaches.com/2016/9/my-workflow-org-agenda/][Gregory Stein]] (GS) and in turn by [[http://doc.norang.ca/org-mode.html][Bernt Hansen]] (BH). You should check out those great resources.

* Design Notes

- Capture templates must be defined *outside* ~with-eval-after-load~
- Use ~expand-file-name~ (or helper) for all paths
- Use backquote + comma when embedding evaluated paths
- Avoid mixing ~:clock-in nil~ with ~:clock-resume t~
- Some completion backends (e.g. citar) must be disabled in capture buffers

* Core configuration (no org dependency)

These settings do not depend on Org being loaded and should be defined eagerly.

** Function definitions

   Function definitions for capture templates and other org-related quality of life adjustments go here.

   #+begin_src emacs-lisp :exports none
     (defun sprazza/org-file (f)
       "Construct paths to org files. Simplifies construction and is more readable."
       (expand-file-name f org-directory))

     (defun sprazza/org-get-target-headline (&optional targets prompt)
       "Prompt for a location in an org file and jump to it."
       (let ((org-refile-targets (or targets org-refile-targets))
             (prompt (or prompt "Capture Location")))
         (org-refile t nil nil prompt)))

     (defun sprazza/disable-citar-in-capture ()
       "Disable citar completion during capture to avoid CAPF errors"
       (remove-hook 'completion-at-point-functions #'citar-capf t))

     (defun sprazza/org-search ()
       "Search in the org directory with `rg'."
       (interactive)
       (spacemacs/compleseus-search nil (expand-file-name org-directory))
       )

     ;; Ensure spacemacs has org-search available more broadly
     (spacemacs/set-leader-keys
       "aos" 'sprazza/org-search
       )
   #+end_src

* Org runtime configuration (after load)

  These settings require Org to be fully loaded.

** General org settings

  #+begin_src emacs-lisp :exports none
    (with-eval-after-load 'org

      ;; Hooks
      (add-hook 'org-mode-hook #'visual-line-mode)
      (add-hook 'org-mode-hook (lambda () (whitespace-mode -1)))
      (add-hook 'org-mode-hook #'flyspell-mode)

      ;; Disable citar in capture buffers
      (add-hook 'org-capture-mode-hook #'sprazza/disable-citar-in-capture)

      ;; Refile setup
      (setq org-refile-targets '((nil :maxlevel . 2)
                                 (org-agenda-files :maxlevel . 3))
            org-refile-use-outline-path 'file
            org-outline-path-complete-in-steps nil
            org-refile-allow-creating-parent-nodes 'confirm)

      ;; Todo keywords
      (setq org-todo-keywords
            '((sequence "TODO(t)" "|" "DONE(d)")
              (sequence "TODAY(T)" "|" "DONE(d)")
              (sequence "PROJECT(p)" "|" "DONE(d)" "CANCELLED(c)")
              (sequence "WAITING(w)" "|")
              (sequence "|" "CANCELLED(c)")
              (sequence "SOMEDAY(s)" "|" "CANCELLED(c)")
              (sequence "|" "MEETING")))

      ;; Tags
      (setq org-tag-alist
            '(("meeting" . ?m)
              ("review" . ?r)
              ("analysis" . ?a)
              ("statistical" . ?s)
              ("documenting" . ?d)
              ("coding" . ?c)))

      (setq
       ;; Triggers for state changes
       org-todo-state-tags-triggers
       (quote (
    	         ;; Move to cancelled adds the cancelled tag
    	         ("CANCELLED" ("CANCELLED" . t))
    	         ;; Move to waiting adds the waiting tag
    	         ("WAITING" ("WAITING" . t))
    	         ;; Move to a done state removes waiting/cancelled
    	         (done ("WAITING") ("CANCELLED"))
    	         ("DONE" ("WAITING") ("CANCELLED"))
    	         ;; Move to todo, removes waiting/cancelled
    	         ("TODO" ("WAITING") ("CANCELLED"))
    	         )
    	        )
       )

      ;; Ensure child dependencies complete before parents can be marked complete
      (setq org-enforce-todo-dependencies t
       )

      ;; For reporting, ensure some good defaults for efforts and clocksums
      (setq org-columns-default-format "%50ITEM(Task) %10Effort(Effort){:} %10CLOCKSUM"
            org-global-properties (quote (("Effort_ALL" . "0:15 0:30 1:00 2:00 4:00 6:00 8:00 16:00")))
            ;; When there's 0 time spent, remove the entry
            org-clock-out-remove-zero-time-clocks t
            )

      )
  #+end_src
